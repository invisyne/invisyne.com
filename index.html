Products

Four products. One continuous machine memory.

Capture

The machine's memory.

Captures at the source — Siemens S7, Modbus, OPC UA, MQTT — up to 5,000 variables per second at 50 ms. Local-first, with no cloud dependency.

Platform

Manage the fleet.

A secure web platform for the whole Edge fleet — device status, firmware, users and roles, dashboards and alerts. The command center for cross-machine intelligence.

Open Hub ↗
Analysis

From symptom to cause.

Root-cause analysis on dense, high-frequency time series. Compare Runs — batches, shifts, periods — across machines to turn data into answers.

On-site

On-site control.

A Windows app to discover, configure, update and export from Edge devices — entirely on the local network, no cloud needed. Built for field and commissioning engineers.
